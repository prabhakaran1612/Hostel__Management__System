# 🏨 Hostel Hub – AI-Powered Hostel Management System

Hostel Hub is a full-stack hostel management system developed using **Python (Flask)** and **MySQL** to simplify hostel administration. The system provides dedicated portals for **Students** and **Wardens**, enabling efficient management of room allocation, complaints, attendance, leave requests, mess feedback, and announcements.

The project integrates AI-powered features such as **smart room allocation**, **complaint priority prediction**, **sentiment analysis for mess feedback**, an **AI chatbot**, and an **analytics dashboard** to enhance decision-making and improve the overall hostel experience.

## ✨ Key Features
- 🔐 Student & Warden Authentication
- 🏠 Smart Room Allocation
- 🤖 AI Complaint Priority Prediction
- 💬 AI Chatbot
- 📅 Leave Management
- 📊 Attendance Tracking
- 🍽️ Mess Menu & Feedback Sentiment Analysis
- 📢 Announcements
- 📈 Analytics Dashboard with Charts
- 🛡️ Secure Role-Based Access Control

## 🛠️ Tech Stack
- **Backend:** Python, Flask
- **Frontend:** HTML, CSS, JavaScript
- **Database:** MySQL
- **AI Modules:** TextBlob, Rule-Based NLP
- **Visualization:** Chart.js

This project demonstrates the application of AI and web technologies to create an intelligent, user-friendly hostel management solution.
